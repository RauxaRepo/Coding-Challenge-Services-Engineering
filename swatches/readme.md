# Swatches

This directory contains sample swatch images you can use for your prototype. 

## Swatch Data Model 

```javascript
{
    "active": Boolean
    "name": String,
    "price": String,
    "image": String,
    "color": String
    "date": String
}
```
### Example
```javascript
{
    "active": true,
    "name": "My Swatch",
    "price": "$10",
    "image": "url to image",
    "color": "#467654",
    "date": "date created"
}
```
